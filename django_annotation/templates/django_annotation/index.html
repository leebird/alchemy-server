miRTex is a text-mining system that extracts miRNA-gene regulation relations and miRNA-target relations from literature. This website hosts the results produced by miRTex. Currently the database stores 7,737 PubMed abstracts with 15,991 unique miRNA-gene regulation pairs and 8,808 unique miRNA-target pairs. We are working on incoporating the miRTex results for the full-length articles from the PMC Open Access Subset.
You can search for a gene, e.g., "BAD", or multiple genes connected by "OR" operator. The query will be sent to PubMed to obtain relevant PMIDs so that PubMed-style query can be used here. Besides, you can also search for a list of PMIDs separated by comma or space, e.g., "22587342, 23981989".
The development and testing corpus used by miRTex can be viewed and downloaded at here.
